ESTRUTURA DO HTML





A estrutura básica do HTML é formada por um conjunto de tags e inicia-se obrigatoriamente com o Doctype que é um código que define ao navegador e aos demais programas que podem ler seu site que o código encontrado ali é um código que utiliza a versão mais atual do HTML. A seguir seguem as principais tags que formam a estrutura do HTML:

A tag html - onde delimitamos a linguagem que usaremos em nosso site através do recurso “lang”. A tag html é o que delimita o escopo de um HTML.

A tag head - contém informações de configuração, ou seja, não são visíveis ao usuário. Na prática, todo o conteúdo do cabeçalho fica delimitado entre a abertura e fechamento tag head. Alguns exemplos são: Configuração de título da aba do site, linkar arquivos externos e etc.

A tag title - é visível ao usuário e aparece na aba da página do site. Esta tag está inserida dentro da tag head.

A tag body - Antes de iniciar a tag body, fechamos a tag head. Na tag body é onde se encontra todo o conteúdo do site, ou seja, todas as informações contidas nesta tag são visíveis ao usuário. Dentro dela podemos utilizar as tags abaixo descritas:

As tags h1 - h6 seguem uma hierarquia que serve como um rankeamento, onde a tag h1 recebe maior importância, ou seja, a h6 recebe menor importância entre elas.

A tag hr cria uma barra horizontal na página do site. É uma tag autocontida, ou seja, precisa ser fechada ao iniciarmos, assim usamos de boa prática. Assim como a hr, temos a tag br que também é uma tag autocontida. A tag br usamos para uma quebra de linha, usada para pular uma linha.

A tag p usamos para parágrafos. Esta tag é um elemento de bloco.

A tag a é uma tag de âncora, serve para links. É um elemento de linha, preenche somente a linha mencionada na tag. Podemos usá-la para linkar uma palavra no meio do texto. Ela acompanha o href que é um atributo que serve para direcionar o usuário quando ele clicar no link.

A tag img serve para imagens. O src é o link daquela imagem e o alt serve para descrever aquela imagem, muito importante para a acessibilidade, por exemplo. Uma observação sobre o link da imagem é que salvando a imagem na pasta img do VSCode utilizamos de boa prática visto que a imagem está salva em nosso código. Usando através de um link diretamente da internet, nosso código fica exposto pois esta imagem pode um dia não estar mais acessível e cause uma interferência em nosso site.

É importante se atentar ao fechamento das tags após as informações inseridas. Fechamos a tag body e a tag html ao finalizarmos nosso código.
